## 题目清单

### 岛屿的周长总和

**题目描述**

一个 m×n 的网格中，1 表示陆地，0 表示海洋。陆地由若干个岛屿组成（相邻的 1 视为同一岛屿，相邻指上下左右）。每个岛屿的 “周长” 定义为：该岛屿所有陆地格子中，与海洋（或网格边界）相邻的边的总数。 请计算所有岛屿的周长总和。



**输入格式**

第一行输入 m, n（1 ≤ m,n ≤ 10^3）。

接下来 m 行，每行 n 个整数（0 或 1）。



**输出格式**

所有岛屿的周长总和。



**输入输出样例**

**输入 #1**

```
3 3
1 0 0
1 1 0
0 1 0
```

**输出 #1**

```
10
```



**输入 #2**

```
2 4
1 1 0 1
1 0 1 1
```

**输出 #2**

```
16
```



### 消防局选址问题

**题目描述**

某个城镇有 n 个街区（编号 0 至 n-1），街区之间通过双向道路连接，每条道路有确定的长度（正整数，单位：米），所有街区和道路构成一个带权无向图。为提升应急响应效率，城镇计划修建 1 座消防局，要求消防局所在的街区到其他所有街区的最长距离（即该街区到所有其他街区的最短路径中的最大值）尽可能小。若存在多个街区满足此条件，选择编号最小的那个。

请你根据街区之间的道路信息，确定最优的消防局选址。



**输入格式**

第一行包含一个整数 n（1 ≤ n ≤ 100），表示街区的数量。

接下来 n 行，每行包含 n 个整数，构成一个 n×n 的邻接矩阵 graph，其中：

graph[i][j] 表示街区 i 与街区 j 之间直接道路的长度； 若 i = j，则 graph[i][j] = 0； 若街区 i 与 j 之间无直接道路，则 graph[i][j] = 0（注意：i ≠ j 时，0 表示无直接道路，而非长度为 0）；



**输出格式**

输出一个整数，表示最优消防局选址的街区编号。



**输入输出样例**

**输入 #1**

```
3
0 1 3
1 0 2
3 2 0
```

**输出 #1**

```
1
```



**输入 #2**

```
4
0 2 0 5
2 0 3 0
0 3 0 1
5 0 1 0
```

**输出 #2**

```
1（助教给的是2，应该写错了）
```



### 连接所有城市的最低成本

**题目描述**

有 n 个城市（编号 0 至 n-1），其中一些城市之间可以修建双向公路。每条公路有对应的修建成本（正整数）。现在需要通过修建公路，使所有城市都能相互连通（即任意两个城市之间都存在一条路径），且修建的总费用最低。请计算这个最低的总费用。如果无论论如何修建公路都无法使所有城市连通（即原图是不连通的），则输出 -1。



**输入格式**

第一行包含两个整数 n 和 m，分别表示城市数量和可修建的公路数量。其中：

1 ≤ n ≤ 1000（城市数量范围）

0 ≤ m ≤ 10^4（公路数量范围）

接下来 m 行，每行包含三个整数 a, b, c，表示：

a 和 b 是两个不同城市的编号（0 ≤ a, b < n） c 是修建这两个城市之间公路的成本（1 ≤ c ≤ 100）



**输出格式**

输出一个整数，表示使所有城市连通的最低总修建成本；若无法连通所有城市，输出 -1。



**输入输出样例**

**输入 #1**

```
3 3  
0 1 1  
1 2 2  
0 2 4  
```

**输出 #1**

```
3
```



**输入 #2**

```
4 2  
0 1 1  
2 3 2  
```

**输出 #2**

```
-1
```
